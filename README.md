# TypeScript Practice & OOP Concepts

This repository contains TypeScript practice problems and Object-Oriented Programming (OOP) examples. It is designed to improve understanding of TypeScript fundamentals, type safety, and core OOP principles.

---

# 🎯 Purpose

This project is created for:

- Strengthening TypeScript fundamentals
- Practicing real-world coding problems
- Understanding OOP concepts in TypeScript
- Writing type-safe and scalable code


---

# 📌 Topics Covered

## TypeScript Fundamentals
- Basic types
- Union types
- Generics
- Interfaces
- Type narrowing
- `any` vs `unknown`

## Object-Oriented Programming (OOP)
- Inheritance
- Encapsulation
- Abstraction
- Polymorphism

## Problem Solving Practice
- Array manipulation
- String operations
- Object handling
- OOP concepts
